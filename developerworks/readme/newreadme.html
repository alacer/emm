IBM developerworks author package readme

Hello, authors. We at developerWorks are looking forward to working with you to publish your content. You may write your content using this author package and an XML editor or a plain text editor.

This author package contains the following:

LocationDescription 
/
 The root directory of the author package (developerworks/) contains the set of scripts you can use to create new articles from templates. Start here. 
readme
 The 
readme
subdirectory contains this readme file.
schema
 These are the schema against which XML articles for developerWorks are validated. V6 and V7 schema are included in the current author package. 
tools
 The 
tools
directory contains tools used by the creation scripts, including scripts, Java, and the templates themselves.		 
web
 The 
web
directory contains scripts and graphics and other "includes" that make it possible to preview a page on your system.		 
xsl
 This is the XSL that actually transforms article XML into HTML and PDF previews on your system.

Learn how to use this author package in the article Authoring with the developerWorks templates.
